Research
Multi-level image security using elliptic curve and magic matrix with advanced encryption standard.
Deril Raju
Committee: (Advisor) Dr. RK Sinha, Manoj Kumar Mukul, Priyank Saxena.
paper / slides

We implement a new magic matrix technique coupled with standard Elliptic curve and AES cryptographic methods to create a new algorithm for image cryptography. The algorithm uses these 3 stages to encrypt and decrypt the image. Each block is carefully selected checking the trade off between other blocks, thus acheiving better accuracy than select papers which have worked on the same topic. Several parameters were used in estimating the goodness of the algorithm like NPCR, UACI, Entropy and other anlysis were done such as key sensitivity analysis to check the security of the model. This paper was published in Multimedia Tools and Applications, an Internationl journal which has an Impact Factor IF of 2.75 (2022).
